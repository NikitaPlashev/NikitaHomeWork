С PHP 5.4 стало возможным прямое разыменование массива, возвращаемого в качестве результата вызова функции или метода. Раньше приходилось использовать временные переменные.
С PHP 5.5 возможно разыменовывать массив буквально.
Пример #7 Разыменование массива
<?php
function getArray() {
    return array(1, 2, 3);
}
// в PHP 5.4
$secondElement = getArray()[1];
// ранее делали так
$tmp = getArray();
$secondElement = $tmp[1];
// или так
list(, $secondElement) = getArray();
?>
Замечание:
Попытка доступа к неопределенному ключу в массиве - это то же самое, что и попытка доступа к любой другой неопределенной переменной: будет сгенерирована ошибка уровня E_NOTICE, и результат будет NULL.

Замечание:
Массив разыменовывающий скалярную величину не являющуюся строкой (string), отдаст NULL без какого либо оповещения об ошибке.


Функции для работы с массивом

array — Создает массив
asort — Сортирует массив, сохраняя ключи
foreach  —  цикл перебора масивов
range — Создает массив, содержащий диапазон элементов
sort — Сортирует массив по элементам, ключи перезаписывает
count — Подсчитывает количество элементов массива или что-то в объекте
extract — Импортирует переменные из массива в текущую таблицу символов
implode  — Объединяет элементы массива в строку
unset - удаляет массив или элемент массива
array_values - возвращает все значения массива в виде массива
array_keys - возвращает все ключи массива в виде массива
key — Выбирает ключ из массива
krsort — Сортирует массив по ключам в обратном порядке
ksort — Сортирует массив по ключам
list — Присваивает переменным из списка значения подобно массиву
array_chunk — Разбивает массив на части
array_key_exists — Проверяет, присутствует ли в массиве указанный ключ или индекс
array_push — Добавляет один или несколько элементов в конец массива
array_search — Осуществляет поиск данного значения в массиве и возвращает ключ первого найденого элемента в случае удачи
array_splice — Удаляет часть массива и заменяет её чем-нибудь ещё
